# SDK Tasks

A collection of MSBuild tasks for the .NET SDK build pipeline.

## Categories

- **Analysis** - Project analysis and reference scanning
- **Build** - Core build operations and path resolution
- **Compatibility** - Legacy compatibility support
- **Compilation** - Source file processing and compilation support
- **Configuration** - Build configuration and environment management
- **Diagnostics** - Logging and diagnostic output
- **Packaging** - NuGet package validation
- **Resources** - Resource file handling
- **Tools** - External tool integration

## Building

```bash
dotnet build src/SdkTasks/SdkTasks.csproj
```
